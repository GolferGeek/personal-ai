# Personal AI Backend

This is the NestJS backend for the Personal AI project, handling agent registry, orchestration, and MCP integration.

## Installation

```bash
npm install
```

## Running the application

### Development mode

For development, you can use one of the following methods:

1. Using Nest CLI directly:
   ```bash
   npx nest start --watch
   ```

2. Or compile and run with Node:
   ```bash
   npm run build
   npm run start:prod
   ```

### Production mode

For production:

```bash
npm run build
npm run start:prod
```

## API Endpoints

- `GET /api/agents` - List all available agents
- `GET /api/agents/:agentId` - Get details for a specific agent
- `POST /api/agents/:agentId` - Execute a specific agent
- `POST /api/orchestrate` - Orchestrate requests
- `GET /api/mcp/sse` - SSE endpoint for MCP
- `POST /api/mcp/messages` - Endpoint for MCP messages
